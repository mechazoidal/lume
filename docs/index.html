Module lume

A collection of functions for Lua, geared towards game development.

weightedchoice (t)
Takes the argument table t where the keys are the possible choices and the value is the choice's weight. A weight should be 0 or above, the larger the number the higher the probability of that choice being picked. If the table is empty, a weight is below zero or all the weights are 0 then an error is raised. Returns either "cat" or "dog" with "cat" being twice as likely to be chosen.

Parameters:

  • t table

Returns:

    random value from t

Raises:

If t is empty

Usage:

    lume.weightedchoice({ ["cat"] = 10, ["dog"] = 5, ["frog"] = 0 })

Creates a wrapper function which calls each supplied argument in the order they were passed to lume.combine(); nil arguments are ignored. The wrapper function passes its own arguments to each of its wrapped functions when it is called.
